# Applied Data Science Capstone — SpaceX Falcon 9 Landing Prediction

IBM Data Science Professional Certificate Capstone Project.

Predict whether the Falcon 9 first stage will land successfully, using data collected from the SpaceX API, web scraping, and SQL queries. Includes interactive dashboards, geospatial analysis, and machine learning models.

## Contents

| File | Description |
|------|-------------|
| `jupyter-labs-spacex-data-collection-api.ipynb` | Data collection via SpaceX REST API |
| `jupyter-labs-webscraping.ipynb` | Web scraping from Wikipedia |
| `labs-jupyter-spacex-Data wrangling.ipynb` | Data cleaning and feature engineering |
| `jupyter-labs-eda-sql-coursera_sqllite.ipynb` | EDA using SQLite queries |
| `edadataviz.ipynb` | Exploratory Data Analysis with visualizations |
| `Interactive Visual Analytics with Folium.ipynb` | Geospatial analysis with Folium maps |
| `Hands-on Lab: Build an Interactive Dashboard with Plotly Dash.md` | Dashboard guide |
| `spacex-dash-app.py` | Plotly Dash interactive dashboard |
| `SpaceX_Machine Learning Prediction_Part_5.ipynb` | ML classification models (SVM, Decision Tree, Logistic Regression, KNN) |
| `spacex_launch_dash.csv` | Cleaned dataset for dashboard |
| `spacex_web_scraped.csv` | Scraped dataset from Wikipedia |
| `ds-capstone-template-coursera.pdf` | Capstone report template |
